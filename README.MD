## 📌 Overview

This repository contains a **production-ready HTML dashboard** for the African Centre for Statistics (ACS) at the United Nations Economic Commission for Africa (UNECA). The solution provides **real-time monitoring, predictive analytics, automated reporting, and comprehensive operational dashboards** — all powered by synthetic datasets to ensure privacy and confidentiality.

### Key Features

| Feature | Description |
|---------|-------------|
| **Real-Time Monitoring** | Live data ingestion visualization with automatic updates every 5 seconds |
| **Predictive Analytics** | ML-based forecasting (Holt-Winters) and anomaly detection (Isolation Forest) |
| **Automated Reporting** | HTML report generation with KPI summaries and actionable insights |
| **Interactive Dashboards** | Chart.js visualizations for trends, forecasts, and alerts |
| **Privacy Compliance** | 100% synthetic dataset – no real or confidential information |
